Mathematical expression of the ideas

Scaling of the parameter vector is quite straightforward. If we have a set of parameters x and a scaling vector s, the scaled parameters q have elements q[i] = x[i] / s[i]

By creating the diagonal matrix S from the vector s, we have

q = x * S-1 and x = q * S
